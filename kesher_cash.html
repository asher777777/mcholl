    ברוכים הבאים ל-API של חברת קשר

    תיעוד זה נועד לסייע לכם לבצע אינטגרציה מלאה עם מערכת התשלומים והניהול של קשר. באמצעות הממשק שלנו תוכלו:

    חסר לכם מידע?

    ה-API שלנו מתפתח כל הזמן. אם אתם זקוקים לנקודת קצה (Endpoint) שאינה קיימת בתיעוד, אנא פנו לצוות התמיכה שלנו. אנו קשובים לצרכי המפתחים ושמחים להרחיב את הגישה למידע בהתאם לבקשתכם.

    איך מתחילים?

    כדי להתחיל לעבוד, יש להנפיק מפתחות גישה דרך מערכת הניהול:

    1. 1
      היכנסו לאתר הניהול של החברה.
    2. 2
      בתפריט הימני, נווטו ל:
      מפתחים
      API למפתחים
      פרטי גישה
    3. 3
      פרטי הגישה (שם משתמש וסיסמה) יישלחו אליכם אוטומטית למייל.

    אופן ההזדהות (Authentication)

    בכל קריאה ל-API (Request), יש לכלול את פרטי הגישה שקיבלתם (שם משתמש וסיסמה) לצורך אימות מול השרת.

    שם שדהסוגתיאור
    userName
    String
    		שם המשתמש ל-API כפי שמופיע בממשק הניהול.
    password
    String
    		הסיסמה המשויכת למשתמש ה-API.

    דגשים חשובים

    • funcשם הפונקציה אותה אתם מבצעים (לדוגמה: SendTransaction).
    • formatמומלץ להגדיר json כדי לקבל תגובה במבנה JSON קריא.
    • Case Sensitivity
      המערכת רגישה לאותיות גדולות וקטנות (Case Sensitive). יש להקפיד על כתיבה מדויקת של שמות השדות והפונקציות.
    • Headers
      בשימוש ב-REST, יש לוודא כי ה-Header מסוג Content-Type מוגדר כ-application/json.
